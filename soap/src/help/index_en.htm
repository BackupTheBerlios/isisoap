I. Introduction

SOAP is a kind of supervisor tool. It provides a graphic environment which displays data about a project in progress. SOAP works with workflow tools, like SOUPEX and OWEP, which provided these data via a web server. The tool also generate statistics on one or many projects.

Supervisor can easily follow projects' evolution.
